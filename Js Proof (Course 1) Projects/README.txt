JavaScript NFT Minting Example
This JavaScript program demonstrates a simple NFT (Non-Fungible Token) minting system. It allows you to create NFT objects with metadata, store them in an array, and perform operations such as listing all minted NFTs and getting the total supply.

Description
This program uses JavaScript to simulate the minting of NFTs, each represented as an object with specific metadata attributes. It includes functions to mint new NFTs, list all minted NFTs with their metadata, and display the total number of minted NFTs.
